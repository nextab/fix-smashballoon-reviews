# Fix Smash Balloon Reviews Cache

## English

Add-on for [Smash Balloon Reviews Feed](https://smashballoon.com/reviews-feed). Syncs posts from the SBR posts table (`sbr_reviews_posts`) into the cache table (`sbr_feed_caches`) when the native SBR cache is built incorrectly.

**Requirements:** WordPress 5.9+, PHP 7.4+, Smash Balloon Reviews Feed (active)

**Installation:** Upload as ZIP or copy folder to `wp-content/plugins/fix-smashballoon-reviews/`, then activate in **Plugins**.

**Usage:** Runs daily via cron. Manual sync: **Settings → SBR Cache Sync** → "Sync now".

**License:** GPL v2 or later

## Deutsch

Ergänzendes Plugin für [Smash Balloon Reviews Feed](https://smashballoon.com/reviews-feed). Synchronisiert die Posts aus der SBR Post-Tabelle (`sbr_reviews_posts`) in die Cache-Tabelle (`sbr_feed_caches`), wenn der native SBR-Cache fehlerhaft aufgebaut wird.

**Voraussetzungen:** WordPress 5.9+, PHP 7.4+, Smash Balloon Reviews Feed (aktiv)

**Installation:** Plugin als ZIP hochladen oder Ordner nach `wp-content/plugins/fix-smashballoon-reviews/` kopieren, dann unter **Plugins** aktivieren.

**Nutzung:** Läuft täglich per Cron. Manuell: **Einstellungen → SBR Cache Sync** → „Jetzt synchronisieren“.

**Lizenz:** GPL v2 oder höher
